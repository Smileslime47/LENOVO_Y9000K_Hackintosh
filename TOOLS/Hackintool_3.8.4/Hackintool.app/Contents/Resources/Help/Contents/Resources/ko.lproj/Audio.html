오디오 패치
  1. 최신 버전의 Lilu + AppleALC를 설치합니다.
  2. config.plist/Devices/Audio에 Inject=NO를 설정합니다.
  3. Clover - DSDT Renames (장치가 있을 경우)
    • AZAL to HDEF
    • HDAS to HDEF
  4. Hackintool에서 Audio 탭을 선택합니다.
  5. Layout ID 목록에서 하나를 고릅니다.
  6. 파일 - 내보내기에서 부트로더 config.plist를 선택합니다. (기존 config.plist 파일을 덮어 씀)
  7. 재시동합니다.
  8. 오디오가 작동하지 않는다면 4번으로 돌아가서 다른 Layout ID를 설정해 보세요.

참고: 모든 Layout ID를 시도했고 그래도 문제가 있다면 패치 - 고급에서 오디오 기기 ID 교체를 시도해 보세요. 그래도 작동하지 않는다면 FakePCIID.kext + FakePCIID_Intel_HDMI_Audio.kext를 설치하세요.
